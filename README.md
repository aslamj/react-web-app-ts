# React Web App TS

### Development
```bash
git clone https://github.com/aslamj/react-web-app-ts.git
cd react-web-app-ts
yarn
yarn start
```

Dev web server runs at [localhost:3000](http://localhost:3000)

### Testing
```bash
yarn jest:coverage
yarn cypress:open
yarn cypress:run
```
